PassGet
PassGet is a password management website that is purely designed for the comfort of internet users. This project focuses on the intricacies of security by displaying multiple webpages that focuses on quality-of-life features that could help internet users, such as a local storage password library, a password generator and lastly a password strength checker that checks and displays the strength and security of the password that was created.
